Code Sample (Python): Quick NPV & IRR


from typing import Iterable

def npv(rate: float, cashflows: Iterable[float]) -> float:
    return sum(cf / (1 + rate)**t for t, cf in enumerate(cashflows))

def irr(cashflows: Iterable[float], guess: float = 0.1, tol: float = 1e-6, max_iter: int = 100) -> float:
    r = guess
    for _ in range(max_iter):
        f = npv(r, cashflows)
        # Numerical derivative (finite difference)
        df = (npv(r + 1e-6, cashflows) - f) / 1e-6
        if abs(df) < 1e-12:
            break
        r_next = r - f / df
        if abs(r_next - r) < tol:
            return r_next
        r = r_next
    return r

# Example
cashflows = [-1000, 300, 420, 680]
print("NPV @ 8%:", round(npv(0.08, cashflows), 2))
print("IRR:", round(irr(cashflows), 4))

Financial Mathematics

Time value of money (compound): \\( \\mathrm{FV} = \\mathrm{PV} \\times (1 + r)^n \\)

Where: \(r\) is the rate per period, \(n\) the number of periods.

Net present value (NPV):

\\[ \\mathrm{NPV} = \\sum_{t=0}^{T} \\frac{C_t}{(1 + r)^t} \\]

Black–Scholes (Call):

\\[ C = S_0 \\, \\Phi(d_1) - K e^{-r T} \\Phi(d_2) \\]
